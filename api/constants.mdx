## Color constants

Pyxel's 16-color palette. Pass these values wherever a color index is expected.

| Constant | Value | Color name | Hex |
| --- | --- | --- | --- |
| `COLOR_BLACK` | 0 | Black | `#000000` |
| `COLOR_NAVY` | 1 | Navy | `#2b335f` |
| `COLOR_PURPLE` | 2 | Purple | `#7e2072` |
| `COLOR_GREEN` | 3 | Green | `#19959c` |
| `COLOR_BROWN` | 4 | Brown | `#8b4852` |
| `COLOR_DARK_BLUE` | 5 | Dark Blue | `#395c98` |
| `COLOR_LIGHT_BLUE` | 6 | Light Blue | `#a9c1ff` |
| `COLOR_WHITE` | 7 | White | `#eeeeee` |
| `COLOR_RED` | 8 | Red | `#d4186c` |
| `COLOR_ORANGE` | 9 | Orange | `#d38441` |
| `COLOR_YELLOW` | 10 | Yellow | `#e9c35b` |
| `COLOR_LIME` | 11 | Lime | `#70c6a9` |
| `COLOR_CYAN` | 12 | Cyan | `#7696de` |
| `COLOR_GRAY` | 13 | Gray | `#a3a3a3` |
| `COLOR_PINK` | 14 | Pink | `#ff9798` |
| `COLOR_PEACH` | 15 | Peach | `#edc7b0` |

```python
pyxel.cls(pyxel.COLOR_BLACK)
pyxel.text(10, 10, "Hello", pyxel.COLOR_WHITE)
```

<Note>
The hex values above are the default palette. You can change individual display colors at runtime by writing to `pyxel.colors[index]` without affecting these constant names.
</Note>

## Graphics resource constants

Limits and dimensions for image banks and tilemaps.

| Constant | Value | Description |
| --- | --- | --- |
| `NUM_COLORS` | 16 | Number of palette colors |
| `NUM_IMAGES` | 3 | Number of image banks (0–2) |
| `IMAGE_SIZE` | 256 | Image bank width and height in pixels |
| `NUM_TILEMAPS` | 8 | Number of tilemaps (0–7) |
| `TILEMAP_SIZE` | 256 | Tilemap width and height in tiles |
| `TILE_SIZE` | 8 | Tile size in pixels |
| `FONT_WIDTH` | 4 | Built-in font glyph width in pixels |
| `FONT_HEIGHT` | 6 | Built-in font glyph height in pixels |

```python
# Iterate over all image banks
for i in range(pyxel.NUM_IMAGES):
    pyxel.images[i].cls(0)

# Compute text position using built-in font metrics
text_x = (pyxel.width - len(label) * pyxel.FONT_WIDTH) // 2
```

## Audio tone constants

Wave shapes available for sound synthesis.

| Constant | Value | Description |
| --- | --- | --- |
| `TONE_TRIANGLE` | 0 | Triangle wave — soft, mellow sound |
| `TONE_SQUARE` | 1 | Square wave — classic chiptune sound |
| `TONE_PULSE` | 2 | Pulse wave — thinner square variant |
| `TONE_NOISE` | 3 | Noise — percussion and effects |

```python
pyxel.sounds[0].set_tones("TSSP")  # triangle, square, square, pulse
```

## Audio effect constants

Per-note effects that modify how a note decays over time.

| Constant | Value | Description |
| --- | --- | --- |
| `EFFECT_NONE` | 0 | No effect |
| `EFFECT_SLIDE` | 1 | Slide pitch from the previous note |
| `EFFECT_VIBRATO` | 2 | Oscillate pitch rapidly |
| `EFFECT_FADEOUT` | 3 | Fade volume to zero over the full note |
| `EFFECT_HALF_FADEOUT` | 4 | Fade volume to zero over half the note |
| `EFFECT_QUARTER_FADEOUT` | 5 | Fade volume to zero over a quarter of the note |

```python
pyxel.sounds[0].set_effects("NFVN")  # none, fadeout, vibrato, none
```

## Key constants

Pass key constants to `pyxel.btn()`, `pyxel.btnp()`, and `pyxel.btnr()`.

```python
if pyxel.btn(pyxel.KEY_LEFT):
    player_x -= 2
```

### Special keys

| Constant | Description |
| --- | --- |
| `KEY_UNKNOWN` | Unknown key |
| `KEY_RETURN` | Enter / Return |
| `KEY_ESCAPE` | Escape |
| `KEY_BACKSPACE` | Backspace |
| `KEY_TAB` | Tab |
| `KEY_SPACE` | Space bar |
| `KEY_NONE` | No key (used to disable the quit key in `pyxel.init()`) |

### Letter keys


## Gamepad constants

Pyxel supports up to four gamepads. Each has an identical set of constants with the prefix `GAMEPAD1_`, `GAMEPAD2_`, `GAMEPAD3_`, or `GAMEPAD4_`.

Use analog axis values with `pyxel.btnv()` and digital buttons with `pyxel.btn()` / `pyxel.btnp()`.

### Axes (GAMEPAD1 shown)

| Constant | Description |
| --- | --- |
| `GAMEPAD1_AXIS_LEFTX` | Left stick horizontal |
| `GAMEPAD1_AXIS_LEFTY` | Left stick vertical |
| `GAMEPAD1_AXIS_RIGHTX` | Right stick horizontal |
| `GAMEPAD1_AXIS_RIGHTY` | Right stick vertical |
| `GAMEPAD1_AXIS_TRIGGERLEFT` | Left trigger (analog) |
| `GAMEPAD1_AXIS_TRIGGERRIGHT` | Right trigger (analog) |

### Buttons (GAMEPAD1 shown)

| Constant | Description |
| --- | --- |
| `GAMEPAD1_BUTTON_A` | A button |
| `GAMEPAD1_BUTTON_B` | B button |
| `GAMEPAD1_BUTTON_X` | X button |
| `GAMEPAD1_BUTTON_Y` | Y button |
| `GAMEPAD1_BUTTON_BACK` | Back / Select |
| `GAMEPAD1_BUTTON_GUIDE` | Guide / Home |
| `GAMEPAD1_BUTTON_START` | Start |
| `GAMEPAD1_BUTTON_LEFTSTICK` | Left stick click |
| `GAMEPAD1_BUTTON_RIGHTSTICK` | Right stick click |
| `GAMEPAD1_BUTTON_LEFTSHOULDER` | Left shoulder (LB) |
| `GAMEPAD1_BUTTON_RIGHTSHOULDER` | Right shoulder (RB) |
| `GAMEPAD1_BUTTON_DPAD_UP` | D-pad up |
| `GAMEPAD1_BUTTON_DPAD_DOWN` | D-pad down |
| `GAMEPAD1_BUTTON_DPAD_LEFT` | D-pad left |
| `GAMEPAD1_BUTTON_DPAD_RIGHT` | D-pad right |

Replace `GAMEPAD1_` with `GAMEPAD2_`, `GAMEPAD3_`, or `GAMEPAD4_` for additional controllers.

```python
# Move with left stick on gamepad 1
axis_x = pyxel.btnv(pyxel.GAMEPAD1_AXIS_LEFTX)
axis_y = pyxel.btnv(pyxel.GAMEPAD1_AXIS_LEFTY)
player_x += axis_x // 8000
player_y += axis_y // 8000

# Jump with the A button
if pyxel.btnp(pyxel.GAMEPAD1_BUTTON_A):
    player_dy = -6
```
