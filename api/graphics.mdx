## Variables

<ParamField path="colors" type="list[int]">
  List of the palette display colors. Specified by 24-bit numerical value. Supports Python list operations.

  ```python
  old_colors = list(pyxel.colors)
  pyxel.colors[15] = 0x112233
  ```
</ParamField>

<ParamField path="images" type="list[Image]">
  List of the image banks (instances of the Image class) (0-2).

  ```python
  pyxel.images[0].load(0, 0, "title.png")
  ```
</ParamField>

<ParamField path="tilemaps" type="list[Tilemap]">
  List of the tilemaps (instances of the Tilemap class) (0-7).
</ParamField>

### Advanced variables

<ParamField path="screen" type="Image">

<ParamField path="cursor" type="Image">

## Drawing functions

### clip

Set the drawing area of the screen from (x, y) with a width of w and a height of h. Call `clip()` with no arguments to reset to full screen.

<ParamField path="x" type="float">
  X coordinate of the upper-left corner.
</ParamField>

<ParamField path="y" type="float">
  Y coordinate of the upper-left corner.
</ParamField>

<ParamField path="w" type="float">
  Width of the clipping area.
</ParamField>

<ParamField path="h" type="float">
  Height of the clipping area.
</ParamField>

---

### camera

Set the drawing offset to (x, y). All subsequent drawing operations will be shifted by (-x, -y). Call `camera()` with no arguments to reset the offset to (0, 0).

<ParamField path="x" type="float">
  X coordinate.
</ParamField>

<ParamField path="y" type="float">
  Y coordinate.
</ParamField>

---

### pal

Replace color `col1` with `col2` when drawing. Call `pal()` with no arguments to reset the palette to the initial state.

<ParamField path="col1" type="int">
  Color to replace.
</ParamField>

<ParamField path="col2" type="int">
  Replacement color.
</ParamField>

---

### dither

Apply dithering (pseudo-transparency) when drawing.

<ParamField path="alpha" type="float">
  Opacity in the range 0.0–1.0. 0.0 is fully transparent, 1.0 is fully opaque.
</ParamField>

### blt

Copy the region of size (w, h) from (u, v) of image bank `img` to (x, y). Negative values for w or h flip the image horizontally or vertically. `colkey` sets the transparent color. `rotate` and `scale` apply transformations centered on the copy region.

<ParamField path="x" type="float">
  Destination X.
</ParamField>

<ParamField path="y" type="float">
  Destination Y.
</ParamField>

<ParamField path="img" type="int/Image">
  Image bank number (0-2) or Image instance.
</ParamField>

<ParamField path="u" type="float">
  Source X in the image bank.
</ParamField>

<ParamField path="v" type="float">
  Source Y in the image bank.
</ParamField>

<ParamField path="w" type="float">
  Width. Use a negative value to flip horizontally.
</ParamField>

<ParamField path="h" type="float">
  Height. Use a negative value to flip vertically.
</ParamField>

<ParamField path="colkey" type="int/None">
  Transparent color. If omitted, no transparency.
</ParamField>

<ParamField path="rotate" type="float" default="0">
  Rotation angle in degrees, centered on the copy region.
</ParamField>

<ParamField path="scale" type="float" default="1">
  Scale factor, centered on the copy region.
</ParamField>

---

### bltm

Copy the region of size (w, h) from (u, v) of tilemap `tm` to (x, y). Each tile is 8×8 pixels stored as (image_tx, image_ty). Negative values for w or h flip the image. `colkey` sets the transparent color. `rotate` and `scale` apply transformations centered on the copy region.

<ParamField path="x" type="float">
  Destination X.
</ParamField>

<ParamField path="y" type="float">
  Destination Y.
</ParamField>

<ParamField path="tm" type="int/Tilemap">
  Tilemap number (0-7) or Tilemap instance.
</ParamField>

<ParamField path="u" type="float">
  Source X in the tilemap.
</ParamField>

<ParamField path="v" type="float">
  Source Y in the tilemap.
</ParamField>

<ParamField path="w" type="float">
  Width. Use a negative value to flip horizontally.
</ParamField>

<ParamField path="h" type="float">
  Height. Use a negative value to flip vertically.
</ParamField>

<ParamField path="colkey" type="int/None">
  Transparent color. If omitted, no transparency.
</ParamField>

<ParamField path="rotate" type="float" default="0">
  Rotation angle in degrees, centered on the copy region.
</ParamField>

<ParamField path="scale" type="float" default="1">
  Scale factor, centered on the copy region.
</ParamField>

## Advanced drawing functions

### blt3d

Draw image bank `img` with perspective projection onto the screen rectangle (x, y, w, h).

<ParamField path="x" type="float">
  Destination X.
</ParamField>

<ParamField path="y" type="float">
  Destination Y.
</ParamField>

<ParamField path="w" type="float">
  Display width.
</ParamField>

<ParamField path="h" type="float">
  Display height.
</ParamField>

<ParamField path="img" type="int/Image">
  Image bank number (0-2) or Image instance.
</ParamField>

<ParamField path="pos" type="(float, float, float)">
  Camera position (x, y, z). x and y match 2D coordinates; z is height.
</ParamField>

<ParamField path="rot" type="(float, float, float)">
  Rotation in degrees. rot_x is vertical, rot_y is horizontal, rot_z is tilt.
</ParamField>

<ParamField path="fov" type="float" default="60">
  Field of view in degrees.
</ParamField>

<ParamField path="colkey" type="int/None">
  Transparent color. If omitted, no transparency.
</ParamField>
