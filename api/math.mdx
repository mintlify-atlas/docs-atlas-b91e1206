## Arithmetic

### ceil(x)

Return the smallest integer greater than or equal to `x`.

| Parameter | Type | Description |
| --- | --- | --- |
| `x` | float | Input value |

**Returns:** `int`

### floor(x)

Return the largest integer less than or equal to `x`.

| Parameter | Type | Description |
| --- | --- | --- |
| `x` | float | Input value |

**Returns:** `int`

### clamp(x, lower, upper)

Return `x` clamped between `lower` and `upper`.

| Parameter | Type | Description |
| --- | --- | --- |
| `x` | int/float | Value to clamp |
| `lower` | int/float | Minimum allowed value |
| `upper` | int/float | Maximum allowed value |

**Returns:** `int/float`

```python
speed = pyxel.clamp(speed, -5, 5)
```

### sgn(x)

Return `1` when `x` is positive, `0` when it is `0`, and `-1` when it is negative.

| Parameter | Type | Description |
| --- | --- | --- |
| `x` | int/float | Input value |

**Returns:** `int/float`

```python
dx = pyxel.sgn(target_x - player_x)  # -1, 0, or 1
```

## Geometry

### sqrt(x)

Return the square root of `x`.

| Parameter | Type | Description |
| --- | --- | --- |
| `x` | float | Input value |

**Returns:** `float`

### sin(deg)

Return the sine of `deg` degrees.

| Parameter | Type | Description |
| --- | --- | --- |
| `deg` | float | Angle in degrees |

**Returns:** `float`

### cos(deg)

Return the cosine of `deg` degrees.

| Parameter | Type | Description |
| --- | --- | --- |
| `deg` | float | Angle in degrees |

**Returns:** `float`

### atan2(y, x)

Return the arctangent of `y/x` in degrees.

| Parameter | Type | Description |
| --- | --- | --- |
| `y` | float | Y component |
| `x` | float | X component |

**Returns:** `float`

```python
# Angle from player to target, in degrees
angle = pyxel.atan2(target_y - player_y, target_x - player_x)
dx = pyxel.cos(angle) * speed
dy = pyxel.sin(angle) * speed
```

<Note>
All trigonometric functions work in degrees, not radians. `sin(90)` returns `1.0`.
</Note>

## Random numbers

### rseed(seed)

Set the seed of the random number generator. Use a fixed seed to get reproducible results, or a time-based value for variety.

| Parameter | Type | Description |
| --- | --- | --- |
| `seed` | int | Seed value (non-negative integer) |

### rndi(a, b)

Return a random integer from `a` to `b`, inclusive.

| Parameter | Type | Description |
| --- | --- | --- |
| `a` | int | Minimum value (inclusive) |
| `b` | int | Maximum value (inclusive) |

**Returns:** `int`

### rndf(a, b)

Return a random float from `a` to `b`, inclusive.

| Parameter | Type | Description |
| --- | --- | --- |
| `a` | float | Minimum value (inclusive) |
| `b` | float | Maximum value (inclusive) |

**Returns:** `float`

## Perlin noise

### nseed(seed)

Set the seed for Perlin noise generation.

| Parameter | Type | Description |
| --- | --- | --- |
| `seed` | int | Seed value (non-negative integer) |

### noise(x, y=0, z=0)

Return the Perlin noise value for the given coordinates. The output is a smooth float that varies continuously across space, making it useful for terrain, cloud, and wave effects.

| Parameter | Type | Default | Description |
| --- | --- | --- | --- |
| `x` | float | — | X coordinate |
| `y` | float | `0` | Y coordinate |
| `z` | float | `0` | Z coordinate |

**Returns:** `float`

## Example

The following snippet shows random and noise functions working together to place trees on generated terrain:

```python
import pyxel

SCREEN_W = 160
SCREEN_H = 120

class App:
    def __init__(self):
        pyxel.init(SCREEN_W, SCREEN_H, title="Terrain Demo")

        # Reproducible random seed for consistent level layout
        pyxel.rseed(42)
        pyxel.nseed(42)

        # Build a height map using 1D Perlin noise
        self.heights = [
            int(SCREEN_H * 0.4 + pyxel.noise(x * 0.05) * 30)
            for x in range(SCREEN_W)
        ]

        # Scatter trees at random x positions above the terrain
        self.trees = [
            (pyxel.rndi(0, SCREEN_W - 1), pyxel.rndf(0.5, 1.5))
            for _ in range(12)
        ]

        pyxel.run(self.update, self.draw)

    def update(self):
        if pyxel.btnp(pyxel.KEY_Q):
            pyxel.quit()

    def draw(self):
        pyxel.cls(1)

        # Draw terrain
        for x, ground_y in enumerate(self.heights):
            pyxel.line(x, ground_y, x, SCREEN_H - 1, 3)

        # Draw trees — height varies by the stored scale factor
        for x, scale in self.trees:
            ground_y = self.heights[pyxel.clamp(x, 0, SCREEN_W - 1)]
            tree_h = int(8 * scale)
            pyxel.rect(x - 1, ground_y - tree_h, 3, tree_h, 11)

App()
```
