## Methods

### set

Set the image at (x, y) using a list of hex-digit strings. Each character represents a color index (0-f).

<ParamField path="x" type="int">
  X coordinate.
</ParamField>

<ParamField path="y" type="int">
  Y coordinate.
</ParamField>

<ParamField path="data" type="list[str]">
  Image data as a list of strings. Each character is a hex color index.
</ParamField>

```python
pyxel.images[0].set(10, 10, ["0123", "4567", "89ab", "cdef"])
```

## Drawing on images

Image instances support all drawing functions from the Graphics API. They work identically to the global functions but draw onto the image rather than the screen.

The supported functions are: `cls`, `line`, `rect`, `rectb`, `circ`, `circb`, `elli`, `ellib`, `tri`, `trib`, `fill`, `blt`, `bltm`, `blt3d`, `bltm3d`, `text`, `clip`, `camera`, `pal`, `dither`.

See the [Graphics](/api/graphics) reference for full parameter documentation.
