## Variables

<ResponseField name="mouse_x" type="int">
  The current x position of the mouse cursor.
</ResponseField>

<ResponseField name="mouse_y" type="int">
  The current y position of the mouse cursor.
</ResponseField>

<ResponseField name="mouse_wheel" type="int">
  The current value of the mouse wheel.
</ResponseField>

### Advanced variables

<ResponseField name="input_keys" type="list[int]">
  List of keys input in the current frame.
</ResponseField>

<ResponseField name="input_text" type="str">
  Text input in the current frame.
</ResponseField>

<ResponseField name="dropped_files" type="list[str]">
  List of files dropped in the current frame.
</ResponseField>

## Functions

### `btn`

Return `True` if the key is currently held down, otherwise return `False`.

<ParamField path="key" type="number" required>
  Key code.
</ParamField>

<ResponseField name="returns" type="bool">
  `True` if the key is pressed.
</ResponseField>

---

### `btnp`

Return `True` if the key is pressed in that frame. When `hold` and `repeat` are specified, after holding the key for `hold` frames, return `True` every `repeat` frames.

<ParamField path="key" type="number" required>
  Key code.
</ParamField>

<ParamField path="hold" type="number" default="0">
  Number of frames to hold before repeat starts.
</ParamField>

<ParamField path="repeat" type="number" default="0">
  Repeat interval in frames. If `0`, no repeat.
</ParamField>

<ResponseField name="returns" type="bool">
  `True` if the key was pressed in that frame.
</ResponseField>

---

### `btnr`

Return `True` if the key is released in that frame, otherwise return `False`.

<ParamField path="key" type="number" required>
  Key code.
</ParamField>

<ResponseField name="returns" type="bool">
  `True` if the key was released in that frame.
</ResponseField>

---

### `mouse`

Show the mouse cursor if `visible` is `True`, and hide it if `False`. The cursor position continues to update even when hidden.

<ParamField path="visible" type="boolean" required>
  Show or hide the cursor.
</ParamField>

---

## Advanced functions

<Note>
  The following functions are intended for analog input reading and headless mode input simulation.
</Note>

### `btnv` <sup>Advanced</sup>

Return the analog value of the specified key (e.g., a gamepad axis value).

<ParamField path="key" type="number" required>
  Key code.
</ParamField>

<ResponseField name="returns" type="int">
  Analog value of the key.
</ResponseField>

---

### `set_btn` <sup>Advanced</sup>

Set the press/release state of the specified key. Primarily used for headless mode input simulation.

<ParamField path="key" type="number" required>
  Target key.
</ParamField>

<ParamField path="state" type="boolean" required>
  `True` for press, `False` for release.
</ParamField>

---

### `set_btnv` <sup>Advanced</sup>

Set the analog value of the specified key. Primarily used for headless mode input simulation.

<ParamField path="key" type="number" required>
  Target key.
</ParamField>

<ParamField path="val" type="number" required>
  Analog value to set.
</ParamField>

---

### `set_mouse_pos` <sup>Advanced</sup>

Set the mouse cursor position. Primarily used for headless mode input simulation.

<ParamField path="x" type="number" required>
  X coordinate.
</ParamField>

<ParamField path="y" type="number" required>
  Y coordinate.
</ParamField>

---

### `set_input_text` <sup>Advanced</sup>

Set the text input for the current frame, replacing any existing text. Primarily used for headless mode input simulation.

<ParamField path="text" type="string" required>
  Text input.
</ParamField>

---

### `set_dropped_files` <sup>Advanced</sup>

Set the dropped file list for the current frame, replacing any existing list. Primarily used for headless mode input simulation.

<ParamField path="files" type="string[]" required>
  List of file paths.
</ParamField>

---
