Pyxel is an open-source retro game engine for Python. Inspired by PICO-8 and TIC-80, it gives you a constrained, creative canvas — a 16-color palette, 4 audio channels, and pixel-art tools — so you can focus on making games.

## Get up and running

<Steps>
  <Step title="Install Pyxel">
    Install via pip — requires Python 3.8 or higher:

    ```bash
    pip install -U pyxel
    ```
  </Step>
  <Step title="Write your first game">
    Create a file `hello.py` with this minimal game loop:

    ```python hello.py
    import pyxel

    pyxel.init(160, 120)

    def update():
        if pyxel.btnp(pyxel.KEY_Q):
            pyxel.quit()

    def draw():
        pyxel.cls(0)
        pyxel.text(55, 54, "Hello, Pyxel!", 7)

    pyxel.run(update, draw)
    ```
  </Step>
  <Step title="Run it">
    ```bash
    pyxel run hello.py
    ```

    Press **Q** to quit, or **Esc** at any time.
  </Step>
  <Step title="Explore examples">
    Copy the built-in examples to your current directory and run them:

    ```bash
    pyxel copy_examples
    cd pyxel_examples
    pyxel run 01_hello_pyxel.py
    ```
  </Step>
</Steps>

## Key features

<CardGroup cols={2}>
  <Card title="16-color palette" icon="palette">
    A fixed palette inspired by retro consoles. Swap colors at runtime with `pyxel.pal()` and `pyxel.colors`.
  </Card>
  <Card title="Cross-platform" icon="globe">
    Runs on Windows, Mac, Linux, and the web via WebAssembly — no installation needed for the web version.
  </Card>
  <Card title="Built-in editors" icon="pencil">
    Image, Tilemap, Sound, and Music editors are included. Launch with `pyxel edit`.
  </Card>
  <Card title="4-channel audio" icon="music">
    Define sounds with notes, tones, and effects — or use MML strings and PCM audio files.
  </Card>
  <Card title="Gamepad and mouse input" icon="gamepad">
    Unified input API for keyboard, mouse, and up to 4 gamepads using `btn()`, `btnp()`, and `btnr()`.
  </Card>
  <Card title="Package and distribute" icon="box">
    Bundle your game as a `.pyxapp` file, convert to a standalone executable, or export to HTML.
  </Card>
</CardGroup>
