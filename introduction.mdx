**Pyxel** (/ˈpɪksəl/) is a retro game engine for Python. With simple specifications inspired by retro gaming consoles — displaying only 16 colors and supporting 4 sound channels — you can easily enjoy making pixel-art-style games.

Pyxel is open source under the [MIT License](https://github.com/kitao/pyxel/blob/main/LICENSE) and free to use, including in commercial projects.

## Specifications

Pyxel's constrained design is intentional — the limits focus your creativity and keep the API learnable in a single sitting.

| Specification | Value |
| --- | --- |
| Platforms | Windows, Mac, Linux, Web |
| Language | Python 3.8+ |
| Screen size | Customizable |
| Colors | 16 (fixed palette, swappable at runtime) |
| Image banks | 3 × 256×256 pixels |
| Tilemaps | 8 × 256×256 tiles |
| Audio channels | 4 |
| Sounds | 64 definable sounds |
| Music tracks | 8 (composed from sounds) |
| Input | Keyboard, mouse, gamepad |
| License | MIT |

<Info>
  Pyxel's specifications and APIs are inspired by [PICO-8](https://www.lexaloffle.com/pico-8.php) and [TIC-80](https://tic80.com/). If you've used either of those, you'll feel at home immediately.
</Info>

## 16-color palette

Pyxel uses a fixed 16-color palette. Each color has a named constant you can use directly in your code (e.g., `pyxel.COLOR_RED`).

| Index | Constant | Hex |
|-------|----------|-----|
| 0 | `COLOR_BLACK` | `#000000` |
| 1 | `COLOR_NAVY` | `#2b335f` |
| 2 | `COLOR_PURPLE` | `#7e2072` |
| 3 | `COLOR_GREEN` | `#19959c` |
| 4 | `COLOR_BROWN` | `#8b4852` |
| 5 | `COLOR_DARK_BLUE` | `#395c98` |
| 6 | `COLOR_LIGHT_BLUE` | `#a9c1ff` |
| 7 | `COLOR_WHITE` | `#eeeeee` |
| 8 | `COLOR_RED` | `#d4186c` |
| 9 | `COLOR_ORANGE` | `#d38441` |
| 10 | `COLOR_YELLOW` | `#e9c35b` |
| 11 | `COLOR_LIME` | `#70c6a9` |
| 12 | `COLOR_CYAN` | `#7696de` |
| 13 | `COLOR_GRAY` | `#a3a3a3` |
| 14 | `COLOR_PINK` | `#ff9798` |
| 15 | `COLOR_PEACH` | `#edc7b0` |

Reference colors in code using their index or named constant:

```python
pyxel.text(10, 10, "Hello!", pyxel.COLOR_WHITE)   # named constant
pyxel.rect(10, 10, 20, 20, 11)                     # index — LIME
```

<Tip>
  The palette is customizable at runtime via `pyxel.colors`. You can swap any of the 16 slots to any RGB value without changing your drawing calls.
</Tip>
