- Fixed tilemap editor selection and viewer rendering bugs

## 2.8.8

- Fixed MML audio engine edge cases

- Fixed memory leaks in temporary image, tilemap, and sound objects
- Fixed doc generator stripping HTML tags inside inline code
- Fixed mouse coordinate handling on startup

  Tilemap tile coordinate type changed from `u8` to `u16`. Code that stores or compares raw tile coordinates with a `u8` type will need to be updated.

## 2.4.0

<Warning>
  This release revamps the sound engine and MML syntax. Several API changes are breaking. See the [Migration Guide](/faq#troubleshooting) for a full list of required code changes.
</Warning>

- Renewed sound engine and MML syntax
- Renamed `waveform` field of `Tone` to `wavetable`
- Updated `play` and `playm` to use `sec` (seconds) instead of `tick`
- Updated `play_pos` return value to `(sound_no, sec)`
- Updated `Sound` and `Music` `save` methods to use `sec` instead of `count`
- Added `total_sec` method to `Sound`
- Added new `mml` method to `Sound`; old syntax available via `old_mml`
- Added MML string support directly to the `play` function
- Renamed `excl_*` options to `exclude_*` in `load` and `save`
- Removed `incl_*` options from `load` and `save`

## 2.0.0

- Switched screen rendering to GLSL shaders
- Added `screen_mode` function to change screen rendering type
- Added `dither` function to set dithering type
- Added `images`, `tilemaps`, `channels`, `sounds`, and `musics` as system lists
- Switched to a new resource format based on TOML

<Warning>
  Version 2.0.0 is a major release with numerous breaking changes. Several functions and fields have been renamed or removed, and the resource file format has changed. Review the full changelog entry before upgrading.
</Warning>
